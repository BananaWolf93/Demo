# ML/DL & Agentic AI Applications:

## Purpose:
This repository is a quick demonstration that involves classical Machine Learning, Deep Learning and Agentic AI development to build a strong case for leadership to understand the impact that prototypes may have by underscoring it's potential impactful benefits.

## Overview:
At a high-level, this project comprises of the following tasks in order:

1. Importing a kaggle dataset
2. Creating a juptyer notebook to perform exploratory data analysis, prototyping with various ML/DL models and implementing advanced techniques including automated hyperaparameter tuning, pipelines and column transformers.
3. Comparing the models
4. Creating a separate notebook to define an agentic AI graph with the LangChain/LangGraph framework with read and write tools so a model can read, understand and generate a full professional report for stakeholders and C-level executives.
